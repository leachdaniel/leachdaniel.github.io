SKILLS

SQL Skills

  • Microsoft SQL Server
    When I took my first SQL course, in 2005, I started developing with Microsoft SQL Server. I'm proficient at query tuning and adding indexes when necessary. Most of the applications I've built and maintained rely largely on SQL Server. Some migration projects include:
    • ORACLE application queries converted to T-SQL in SSRS reports
    • HTML content in SQL Server converted to PDFs in AWS S3
  • MySQL
    Most of my involvement with MySQL was earlier on in my career as an implementation specialist at Kaplan when I supported and made specialized reports for B2B clients. I've also supported database transactions that synchronize data in SQL Server and MySQL as well as some read queries and stored procedure calls made by .Net applications.
  • PostgresSQL
    I've converted a Trac Wiki that used PostgresSQL for persistence to GitHub Markdown. I also supported applications that saved and read error logs in PostgresSQL.
